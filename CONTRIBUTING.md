# Contributing

Contributions are welcome! Here's how to get started:

## Setup

```bash
git clone https://github.com/waleedoff/spring-scaffold-maven-plugin.git
cd spring-scaffold-maven-plugin
mvn install
```

## Making Changes

1. Fork the repository
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Build and verify: `mvn clean package`
5. Commit and push your branch
6. Open a Pull Request

## Guidelines

- Keep changes focused and small
- Test your changes against a real Spring Boot project before submitting
- Follow the existing code style
